Filters

These is the list of custom filters added by Lektor in addition to the built in filters.

asseturl

Generates a relative URL from the current page to an asset.

dateformat

Formats a date into a string

datetimeformat

Formats a datetime into a string

latformat

Formats the latitude into a string

latlongformat

Formats the latitude and longitude into a string

longformat

Formats the longitude into a string

markdown

Converts markdown to HTML

tojson

Converts a data structure to JSON.

url

Generates a relative URL from the current page to another.
